# Modeling App

UMLのクラス図のような記法で、テストアーキテクチャを記述するためのアプリケーションです。
Reactで作成されており、Create React Appをベースにしています。

## インストール

このプロジェクトをローカルにインストールするには、以下の手順を実行します。

```sh
git clone <リポジトリのURL>
cd <リポジトリ名>
npm install
```

## 使用方法

```sh
npm start
```

## ビルド
本番環境用にビルドするには、以下のコマンドを実行します。

```sh
npm run build
```

## テスト
テストを実行するには、以下のコマンドを実行します。

```sh
npm test
```
